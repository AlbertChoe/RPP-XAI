# RPP-XAI: Research Project Prototype with Explainable AI

This project implements a machine learning model with explainable AI capabilities for analyzing and interpreting predictions.

## Installation

First, install the required package manager:

```bash
pip install uv
```

## Usage

### To train and save the model

```bash
uv run -m src.train_model
```

### To generate full report

```bash
uv run -m src.generate_full_report
```

## Project Structure
